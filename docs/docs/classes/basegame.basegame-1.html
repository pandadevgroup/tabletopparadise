Class BaseGame<DomHelperType, PlayerType, TabletopType, ServerConnectionType>

BaseGame provides the necessary utilities for a game.

It handles resizing, creating a Tabletop.Tabletop instance, communicating with the server, etc.

Extend this class to create functionality for a custom game. You may override any public/protected method provided you call super().
